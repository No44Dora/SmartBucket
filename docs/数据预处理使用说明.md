# 数据预处理使用说明（彩色平涂图 -> 区域实例标签）

本文对应脚本：`scripts/prepare_dataset.py`。

## 1. 功能概览

脚本会按以下流程处理输入图像：

1. 读取彩色图像；
2. 可选双边滤波平滑（`quality` 模式默认开启）；
3. 在 Lab 空间进行 KMeans 颜色聚类（支持采样训练 + 全图最近中心分配）；
4. 基于 Lab 距离合并相近颜色；
5. 可选剔除线稿/极暗像素（默认开启）；
6. 对每种合并后颜色做形态学清理 + connected components；
7. 为每个连通块分配全局 `region ID`；
8. 导出 `region_id` 与元信息（可选对子集导出可视化）。

## 2. 命令行参数

### 推荐命令（fast 批量模式）

```bash
python scripts/prepare_dataset.py \
  --input data/processed/color_256 \
  --output data/raw/region_labels \
  --mode fast \
  --num-clusters 24 \
  --num-workers 4
```

### 推荐命令（quality 调参模式）

```bash
python scripts/prepare_dataset.py \
  --input data/processed/color_256 \
  --output data/raw/region_labels \
  --mode quality \
  --num-clusters 32 \
  --merge-lab-threshold 12 \
  --min-region-area 128 \
  --save-vis \
  --vis-max-images 20
```

### 参数说明

- `--input`：输入单张图像路径，或图像目录（支持 png/jpg/jpeg/bmp/webp）。
- `--output`：输出目录。
- `--mode`：`fast | quality | custom`。
  - `fast`：默认采样更小、attempts 更低、关闭平滑，适合批量生产。
  - `quality`：采样更大、attempts 更高、开启平滑，适合调参与复杂图。
  - `custom`：完全按其余参数值执行。
- `--num-clusters`：初始聚类中心数。
- `--sample-size`：KMeans 训练采样像素数（仅训练用，预测仍是全图）。
- `--kmeans-attempts`：KMeans 初始化尝试次数。
- `--merge-lab-threshold`：Lab 颜色中心合并阈值（越大越容易并色）。
- `--min-region-area`：连通域最小面积过滤。
- `--num-workers`：并行进程数，`0` 表示串行。
- `--save-vis`：是否输出可视化图。
- `--vis-max-images`：最多可视化多少张图（用于“只可视化部分数据”）。
- `--max-vis-instances`：单图实例可视化最大数量。

## 3. 输出文件说明

对于输入 `xxx.png`，会在输出目录产生（文件前缀为 `xxx__png`，用于避免与 `xxx.jpg` 冲突）：

- `xxx__png_region_id.png`：实例 ID 标签图（`uint16`，写出前会检查溢出风险）；
- `xxx__png_meta.json`：处理参数与统计信息（采样像素数、合并后颜色数、区域数、耗时等）；
- `xxx__png_pipeline.png`：流程总览图（仅在 `--save-vis` 且该图被选中可视化时输出）；
- `xxx__png_instances.png`：部分实例“前后”可视化（同上）。

## 4. 调参建议

- 如果颜色碎片太多：
  - 先增大 `--min-region-area`（如 128 / 256）；
  - 再增大 `--merge-lab-threshold`（如 12 -> 16）。
- 如果不同区域被误合并：
  - 降低 `--merge-lab-threshold`（如 12 -> 8）。
- 如果聚类耗时高：
  - 降低 `--sample-size`；
  - 设置 `--num-workers` 为 4/6/8（按机器资源调节）。
- 如需快速抽查效果：
  - 使用 `--save-vis --vis-max-images N`，仅保存前 N 张可视化。

## 5. 依赖说明

脚本依赖：

- `numpy`
- `opencv-python`（`cv2`）
- `matplotlib`

如出现 `ModuleNotFoundError: No module named 'cv2'`，请先安装 `opencv-python`。
