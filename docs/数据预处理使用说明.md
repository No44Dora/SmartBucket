# 数据预处理使用说明（彩色平涂图 -> 区域实例标签）

本文对应脚本：`scripts/prepare_dataset.py`。

## 1. 功能概览

脚本会按以下流程处理输入图像：

1. 读取彩色图像；
2. 颜色量化 / 聚类（KMeans）；
3. 合并相同或相近颜色（优先按色相，容忍明度/饱和度差异）；
4. 对每种合并后颜色做 connected components；
5. 为每个连通块分配全局 `region ID`；
6. 导出流程可视化与若干实例前后对比。

## 2. 命令行参数

```bash
python scripts/prepare_dataset.py \
  --input <输入图像或目录> \
  --output <输出目录> \
  --num-clusters 24 \
  --hue-threshold 12 \
  --sat-threshold 60 \
  --val-threshold 70 \
  --min-region-area 16 \
  --max-vis-instances 6
```

### 参数说明

- `--input`：输入单张图像路径，或图像目录（支持 png/jpg/jpeg/bmp/webp）。
- `--output`：输出目录。
- `--num-clusters`：初始颜色聚类数量。
- `--hue-threshold`：色相合并阈值（越大越容易并色）。
- `--sat-threshold`：饱和度容忍阈值（应对材质/阴影变化）。
- `--val-threshold`：明度容忍阈值（应对阴影/渐变）。
- `--min-region-area`：过滤过小区域。
- `--max-vis-instances`：单图实例可视化最大数量。

## 3. 输出文件说明

对于输入 `xxx.png`，会在输出目录产生（文件前缀为 `xxx__png`，用于避免与 `xxx.jpg` 冲突）：

- `xxx__png_region_id.png`：实例 ID 标签图（`uint16`）；
- `xxx__png_pipeline.png`：流程总览图（输入、量化、合并、实例 ID 着色）；
- `xxx__png_instances.png`：部分实例“前后”可视化；
- `xxx__png_meta.json`：处理参数与统计信息（颜色数、区域数等）。

## 4. 阈值调参建议

若图像存在阴影、颜色渐变：

- 建议优先保持较小 `hue_threshold`（如 8~15），防止跨色相误合并；
- 适当增大 `sat_threshold` 与 `val_threshold`（如 50~90），提高对阴影/高光容忍度；
- 若细碎噪声区域太多，可增大 `min_region_area`。

## 5. 依赖说明

脚本依赖：

- `numpy`
- `opencv-python`（`cv2`）
- `matplotlib`

如出现 `ModuleNotFoundError: No module named 'cv2'`，请先安装 `opencv-python`。
